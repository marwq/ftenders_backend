## ⚠️ ВАЖНЫЕ ПРИМЕЧАНИЯ

1. **Приложения 9, 11, 12** заполняет ПОСТАВЩИК (внутри документов написано: "заполняется потенциальным поставщиком")
2. **Приложение 19** существует в 2 версиях: банковская гарантия (БАНК) и сведения о субподрядчиках (ПОСТАВЩИК+СУБПОДРЯДЧИК)
3. **Приложение 6** имеет несколько вариантов для разных типов закупок (товары/услуги/аукцион)
4. Субподрядчик должен подтвердить участие через ЭЦП на портале goszakup.gov.kz
